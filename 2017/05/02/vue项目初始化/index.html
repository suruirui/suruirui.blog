使用vue-cli工具初始化项目

安装vue-cli脚手架

npm install -g vue-cli

查看vue版本

vue -V 
vue list

创建项目 vue init 模板名称 项目名称

vue init webpack sales
vue init webpack-simple sales

设置项目描述信息

进入项目目录

cd sales

安装项目依赖

npm install

启动项目

npm run dev

输入地址localhost:8080运行,报错处理 ：

需要 node 版本大于4.0.0 npm版本大于3.0.0
升级npm之后仍然无法启动
需要将C:\Users\用户名\AppData\Roaming\npm\node_modules\npm下的
文件全部复制覆盖到node安装目录下的npm目录下
