在官网下载http://www.yiichina.com/download，选择从归档文件安装，Yii2的基本应用程序模板

将下载文件解压到www目录，访问http://localhost/yii/basic/requirements.php检测安装环境

访问http://localhost/yii/basic/web/index.php执行安装，如果报错，在config目录下的web.php为cookieValidationKey添加任意值

创建第一个控制器：在controllers下面新建HelloController.php:

namespace app\controllers;
use yii\web\Controller;
class HelloController extends Controller{
    public function actionIndex(){
        echo 'hello world';
    }
}

访问创建的HelloController控制器http://localhost/yii/basic/web/index.php?r=hello/index

使用$req = \YII::$app->request; 创建请求对象,调用$req的get方法处理请求参数$id = $req->get(‘id’);

使用$req->isGet来判断请求方式是否为get

->isPost来判断请求方式是否为post

$req->userIp获取用户ip

处理响应，获得响应组件$res = \YII::$app->response;

$res->statusCode = ‘404’

处理响应头

$res->headers->add('pragma','no-cache');
$res->headers->set('pragma','max-age=5');
$res->header->remove('pragma');

跳转

$res->headers->add('location','http://www.baidu.com');

重定向

$this->redirect('http://www.baidu.com',302);

文件下载

$res->headers->add('content-disposition','attachment;filename='a.jpg');
$res->sendFile('./b.jpg');

session使用

得到session对象 $sesssion = \YII::$app->session;

session中方法 set get remove

以数组的方式操作session

$session['username'] = 'lisi';  //set
echo $session['username'];  //get
unset($session['username']);  //remove

cookie使用

视图 views目录

在views目录新建hello(需要与控制器名称保持一致)，新建index.php，直接书写html代码
在控制器HelloController方法中 return $this->renderPartial(‘index’);

视图中的数据传递

传递变量

$data数组
$str = ‘hello yii’;
//创建一个数组
$data = array();
//把需要传递给视图的数据。放到数组当中
$data[‘view_str’] = $str;
return $this->renderPartial(‘index’,$data);
在视图文件中直接使用变量<?=$view_str?>

传递数组

同传递变量，在视图中直接使用数组<?=$arr[1]?>

数据安全,防止转义攻击

use yii\helper\Html;
use yii\helper\HtmlPurifer;
<?=Html::encode($str)?>
<?=HtmlPurifer::process($str)?>
