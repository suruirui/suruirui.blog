sublimetext的markdown插件安装

发表于 2016-09-30

SublimeText markdown插件安装
1.ctrl+shift+P调出PackageControl,输入install package
2.输入markdown editing安装md编辑插件，执行安装
3.再次打开PackageControl,输入install package,输入Markdown Preview，进行安装
Preference -> Markdown Preview -> Settings User加入

    "enable_highlight": true,
    "enable_mathjax": true,

配置markdown高亮显示

4.配置markdown preview编译生成html文件：工具-编译系统-MarkDown
5.使用ctrl+B,生成html文件
6.markdown语法
